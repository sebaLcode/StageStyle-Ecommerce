## Pasos para Ejecutar el Proyecto

1. Clonar Repositorio

2. Esperar que cargue unos segundos

3. Tener instaladas las dependencias necesarias del proyecto, React, Bootstrap, Karma y Jasmine, etc.
   
4. Dentro de la carpeta src, agregar firebase-config.js, que contiene la configuración de Firebase.

5. Ejecutar el Backend (https://github.com/sebaLcode/StageStyle-API) mediante npm run dev.

6. Ejecutar la app con comando npm run start.

7. Para acceder a la pantalla del admin usar correo: admin@stagestyle.com y contraseña: admin1234. Para acceder como vendedor usar correo: vendedor@stagestyle.com y contraseña: vend1234.
